# Heart Disease Detector ❤️

A Machine Learning web app that predicts your risk of heart disease using key health parameters. Built with **Python** and **Streamlit**, it delivers instant results through a clean, interactive interface.

[![Live Demo](https://img.shields.io/badge/Streamlit-Live%20Demo-success?style=for-the-badge&logo=streamlit)](https://heart-disease-detector-rexs5dv4qem4sa5mn784pd.streamlit.app/)

---

## 🩺 What is this?

**Heart Disease Detector** helps you estimate your risk for heart disease based on vital stats like age, cholesterol, blood pressure, chest pain type, ECG data, and more.  
Just input your details and get an instant prediction: **Low Risk** or **High Risk**.

## 🌟 Features

- **Real-time Prediction:** Instant feedback on heart disease risk.
- **Easy to Use:** Clean interface with intuitive controls.
- **Multiple Parameters:** Age, Sex, Chest Pain Type, Blood Pressure, Cholesterol, ECG, Heart Rate, and more.
- **Visual Results:** Color-coded output for quick understanding.
- **Accessible:** Free, open-source, and live demo available.

---

## 📋 How It Works

1. **Enter Health Parameters:**  
   - Age, Sex, Chest Pain Type, Resting Blood Pressure, Cholesterol, Fasting Blood Sugar, Resting ECG, Max Heart Rate, Exercise-Induced Angina, Oldpeak, ST Slope.

2. **Click Predict:**  
   - The app analyzes your data using a trained machine learning model.

3. **View Result:**  
   - Instantly shows your risk:  
     *🟢 Low Risk of Heart Disease*  
     *🔴 High Risk of Heart Disease*

## 📦 Installation

```bash
git clone https://github.com/Ramesh143-code/heart-disease-detector.git
cd heart-disease-detector
pip install -r requirements.txt
streamlit run app.py
```

## 🧑‍💻 Usage

- Enter your details.
